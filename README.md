A modern, responsive personal portfolio built to showcase projects, experience, and skills. The site focuses on accessibility, fast load times, and smooth micro-interactions while using a component-driven architecture so pieces can be reused across pages.

<p align="center">
	<img src="public/unil.png" alt="Website preview - Unil" style="max-width:100%;height:auto;border-radius:12px;box-shadow:0 10px 30px rgba(2,6,23,0.4)" />
</p>

## Features

- Clean, responsive design with Tailwind CSS
- Reusable UI components and lightweight primitives
- Smooth animations and transitions via Framer Motion
- Accessibility-minded markup and keyboard interactions
- Project pages with demo links, tech listed, and screenshots

## Tech Stack

- ![Next.js](https://img.shields.io/badge/Next.js-16-black?style=flat-square&logo=nextdotjs) Next.js (App Router)
- ![TypeScript](https://img.shields.io/badge/TypeScript-5.0-blue?style=flat-square&logo=typescript) TypeScript
- ![React](https://img.shields.io/badge/React-19-61DAFB?style=flat-square&logo=react) React 19
- ![TailwindCSS](https://img.shields.io/badge/Tailwind_CSS-v4.0-38B2AC?style=flat-square&logo=tailwindcss) Tailwind CSS + PostCSS
- ![Framer Motion](https://img.shields.io/badge/Framer_Motion-✓-F24E1E?style=flat-square&logo=framer) Framer Motion
- Radix UI primitives, Tabler & Lucide icons, and other lightweight utilities

## Repo Structure

- `app/` — Next.js app routes and pages
- `components/` — UI components and small primitives (see `components/ui/`)
- `public/` — static assets (logo: `public/favicon.svg`)
- `lib/` — utility helpers
